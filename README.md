# Argus
Regression results aggregator and analyzer

- preserve test runs, test results, test logs and more
- provides critical reports to help identify and fix issues faster


## About the Name
- We were inspired by Argus Panoptes, a many-eyed giant in Greek mythology.
- "Sleep never fell upon his eyes; but he kept sure watch always."
